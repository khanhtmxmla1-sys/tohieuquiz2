# Design Specifications: Footer Green Meadow

## 🎨 Bảng màu (Color Palette)
| Tên màu | Mã Hex / RGBA | Sử dụng |
|---------|---------------|---------|
| **Glass Background** | `rgba(220, 252, 231, 0.7)` | Nền chính Footer |
| **Title Text** | `#064E3B` | Tiêu đề các cột |
| **Body Text** | `#166534` | Văn bản nội dung, links |
| **Link Hover** | `#16a34a` | Màu khi di chuột vào link |
| **Border Glass** | `1px solid rgba(255, 255, 255, 0.3)` | Đường ngăn cách trên |

## 📝 Typography
- **Font Family**: `'Baloo 2', sans-serif` (Import từ Google Fonts nếu chưa có).
- **Heading Size**: `1.125rem` (18px) - Bold 700.
- **Body Size**: `0.95rem` (15.2px) - Semibold 500.

## 📐 Spacing & Effects
- **Backdrop Blur**: `12px`.
- **Vertical Padding**: `48px` (Top/Bottom).
- **Column Gap**: `32px`.
- **Border Radius (Icons)**: `12px`.

## 📱 Mobile Responsiveness
- **Desktop**: Grid 4 cột.
- **Tablet**: Grid 2 cột.
- **Mobile**: Stack 1 cột, căn giữa toàn bộ text.
