# /clean

1. Tải bộ kỹ năng `@code-refactoring-refactor-clean`.
2. Quét file đang mở để tìm các "mùi" của code (code smells), sự trùng lặp hoặc độ phức tạp không cần thiết.
3. Lập kế hoạch tái cấu trúc (Refactor Plan) từng bước nhỏ.
4. Áp dụng các nguyên tắc SOLID và Clean Code để làm sạch mã nguồn.
5. Đảm bảo hành vi của ứng dụng không thay đổi sau khi làm sạch.
