# /senior

1. Tải bộ kỹ năng `@senior-engineering-toolkit`.
2. Phân tích kiến trúc hiện tại của component/page đang mở.
3. Đánh giá chỉ số FFCI (Architectural Fit, Reusability, Performance).
4. Đề xuất cải tiến theo tiêu chuẩn: Suspense-first, Feature-based, Strict TS.
5. Thực hiện thay đổi nếu được yêu cầu.
